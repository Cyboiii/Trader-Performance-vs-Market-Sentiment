# Trader Performance vs Market Sentiment Analysis

## 📌 Overview

This project analyzes how market sentiment (Fear vs Greed) impacts trader behavior and performance using Hyperliquid trading data and Bitcoin Fear/Greed index.

---

## 📊 Objectives

* Compare trader performance across Fear vs Greed conditions
* Analyze behavioral changes (trading frequency, leverage, position size)
* Identify trader segments
* Derive actionable trading strategies

---

## ⚙️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/Cyboiii/Trader-Performance-vs-Market-Sentiment.git
cd Trader-Performance-vs-Market-Sentiment
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the notebook

```bash
jupyter notebook
```

Open:

```
notebook/analysis.ipynb
```

---

## 📈 Outputs

* Charts comparing trader behavior across sentiment
* Aggregated performance metrics
* Segment-based analysis

---

## 🧠 Key Insights

* Traders increase activity during Greed phases
* Risk-taking behavior rises in bullish sentiment
* Fear phases show reduced but more cautious trading

---

## 💡 Strategy Recommendations

* Reduce leverage during Fear periods
* Maintain controlled aggression during Greed periods
